## MerkleTree 全树物化 + leaf_path（已实现）

- 任务：为 Tamper Lab 可视化提供遍历整棵 Merkle 树的能力，不改变 compute_root/verify_inclusion 的已有契约。
- 实现：
  - `compute_tree(leaves) -> MerkleTree?`：物化完整树（所有层级），None 对空输入，奇数节点提升规则与 compute_root 一致（CVE-2012-2459 防御）
  - `MerkleTree.root() / height() / leaf_count() / level(i)`：访问器，越界 level 返回 [] 无 panic
  - `PathStep` / `MerkleTree::leaf_path(index) -> Array[PathStep]?`：从叶子到根脊线，每层级一步，末尾是根；越界返回 None
  - 新增 11 个测试：空树 / 与 compute_root 等价性（8 种形状） / 单叶子 / 双叶子 / 三叶子奇节点提升 / 五叶子 4 层折叠 / 越界 level 安全 / leaf_path 结构校验 + 越界
  - 测试：240→251，wasm-gc 251 passed / js 251 passed，双后端全绿
- 关键设计决策：越界安全（返回 [] 而非 panic）、promoted node 不自我配对、public 仅露出原始不含全路径的 MerkleTree API、`pub(all)` 仅供工作台跨包使用
