创建证据包

拖入一组文件，MoonEvidence 会计算每个文件的 SHA-256 摘要、组装 manifest、计算 Merkle 根，生成可下载的证据包。

Merkle 包含证明

给定一个 manifest 和文件索引，生成该文件的 Merkle 包含证明；也可以验证一个已有的证明。

审计日志（哈希链）

创建一个追加式哈希链审计日志。每条记录的 hash 链接到前一条，任何篡改都会让链条断裂。

Ed25519 数字签名

纯 MoonBit 实现的 Ed25519（RFC 8032），无外部密码学依赖。生成密钥对、签名消息、验证签名——全部在浏览器本地完成。
