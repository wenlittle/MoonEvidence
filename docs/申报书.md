## 项目简介

MoonEvidence 是一个纯 MoonBit 实现的可信证据包验证库和 CLI 工具。给定一组文件及其元数据、SHA-256 摘要、Merkle 树证明和版本链记录，MoonEvidence 能验证这些内容是否完整且未被篡改。

项目采用"链无关验证核心"定位——不是区块链框架，而是一个可在区块链存证前、数据集归档、数字版权打包、AI 输出审计、学术成果发布等场景中复用的通用验证引擎。核心逻辑零 IO 依赖，同一套代码可运行于 CLI、CI 和浏览器三种环境。

## 核心功能

- **Canonical JSON 序列化**：遵循 RFC 8785 规范，确保 JSON 序列化结果字节级稳定，可安全用于摘要计算
- **SHA-256 摘要**：纯 MoonBit 实现，通过 NIST 标准向量验证，吞吐量约 58 MiB/s（js 后端）
- **Merkle 树验证**：RFC 6962 风格域分离，支持根计算与证明验证，与独立 Node 参考实现交叉校验
- **证据清单模型**：带验证的 manifest 解析，拒绝路径遍历攻击，支持文件级摘要比对
- **线性版本链验证**：确保版本记录的时间线性和哈希链完整性
- **7 步验证流水线**：从解析到诊断的完整验证管线，输出结构化 findings 报告
- **CLI 工具**：`verify [--json]` / `explain` 命令，冻结退出码（0 通过 / 1 失败 / 2 错误）
- **浏览器端验证**：同一核心编译为 ESM 模块，支持纯客户端验证，无需上传服务器

## 为什么值得做

MoonBit 生态当前缺少数据完整性验证基础设施。MoonEvidence 填补了这一空白，提供了一个边界清晰、可复用、可发布为 Mooncakes 包的验证核心，同时通过三后端全绿和浏览器 Demo 展示了 MoonBit 的跨平台能力。
